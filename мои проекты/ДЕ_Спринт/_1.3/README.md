*Задание 2*
*Палиндром строки*

Дана строка X, возвращайте True, если X является палиндромом. Пробелы не учитываются.

Гарантируется, что исходная строка может содержать символы только нижнего регистра.


Пример 1:

Ввод: x = “taco cat”

Вывод: true



Пример 2:

Ввод: x = “rotator”

Вывод: true



Пример 3:

Ввод: x = “black cat”

Вывод: false
